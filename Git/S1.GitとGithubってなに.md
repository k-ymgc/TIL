## なぜGitが必要なのか　　
1.「ファイルのバージョン管理をするため」
ファイルのバージョン管理をしないと、どれが最新のドキュメントか分からなくなる。。　　
・ファイル名の後ろに日付をつけて管理。。同じ日付でも名前にコピーがついたものが。。　　
・複数人でファイル更新しようとするとデグレったりする　　
→ファイルのバージョン管理　　
  ・ いつ・誰が・何を変更したか記録を残す　　
  ・ファイルの最新状態がわかる　　
  ・以前の状態にも戻せる
